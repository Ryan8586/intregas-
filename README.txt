Entrega II — Estilização e Layouts
Projeto exemplar que aplica CSS3 avançado, System Design e layouts responsivos.

Conteúdo da pasta:
- index.html
- style.css (design system, grid, flexbox, componentes)
- script.js (interações: menu, modal, toasts, validação)
- README.txt (este arquivo)

Funcionalidades implementadas (conforme especificações):
- Sistema de Design com variáveis CSS (paleta >= 8 cores, tipografia >= 5 tamanhos, espaçamento modular)
- Layout principal com CSS Grid; componentes internos usando Flexbox
- 5 breakpoints responsivos: 1280px, 1024px, 768px, 640px, 480px
- Grid customizado de 12 colunas (utilitários .row/.col-*)
- Menu principal com submenu (dropdown) e menu mobile (hamburguer)
- Componentes: cards responsivos, botões com estados, formulários com validação visual, toasts, modal, badges/tags

Como usar:
1. Extraia o ZIP e abra index.html no navegador.
2. Teste responsividade redimensionando a janela, abra o menu mobile e interaja com o formulário.
